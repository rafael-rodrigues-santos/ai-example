# 🚀 Landing Page de Alta Conversão - O Novo Mercado

Esta é uma Landing Page desenvolvida com foco em performance, experiência do usuário e conversão para o mercado de **Infoprodutos**. O projeto demonstra a aplicação de tecnologias modernas e um fluxo de trabalho acelerado por Inteligência Artificial.

- **Repositório:** [Link Github](https://github.com/rafael-rodrigues-santos/ai-example)

## 🛠️ Stack Técnica
- **Framework:** Next.js 14+ (App Router)
- **Linguagem:** TypeScript
- **Estilização:** Tailwind CSS
- **Componentes:** Shadcn/ui
- **Animações:** Framer Motion

## 🧠 Workflow de IA (Desenvolvimento Aumentado)
Este projeto foi construído utilizando IA como ferramenta central de produção, garantindo agilidade sem perder a qualidade técnica:
1.  **Prototipagem:** Estruturação macro de componentes e layout via **v0.dev**.
2.  **Desenvolvimento:** Uso do **Cursor (Composer)** para refatoração lógica, tipagem de dados e polimento de UI.
3.  **Otimização:** IA aplicada para ajustes de responsividade mobile-first e otimização de performance (Core Web Vitals).

## 📈 Diferenciais e Boas Práticas
- **Acessibilidade (a11y):** Preocupação com semântica HTML, contrastes adequados e suporte para leitores de tela, garantindo que o conteúdo seja acessível a todos os usuários.
- **Performance:** Imagens e fontes otimizadas para carregamento instantâneo.
- **Mobile-First:** Experiência fluida para tráfego vindo de dispositivos móveis.
- **Estrutura de Vendas:** Seções estrategicamente posicionadas (Hero, Social Proof, FAQ e Pricing) para maximizar a conversão.

## 💻 Como rodar o projeto

Siga as instruções abaixo para configurar o ambiente e executar o projeto localmente:

### 1. Pré-requisitos
Certifique-se de ter instalado:
- **Node.js** (Versão 18.0 ou superior)
- **npm** (ou Yarn/pnpm)

### 2. Clonar o Repositório
No seu terminal, execute:
```bash
git clone https://github.com
```

### 3. Instalar dependencias
Entre na pasta do projeto e instale as bibliotecas necessárias:
```bash
cd seu-repositorio
npm install
```

### 4. Executar o Projeto
Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

### 5. Visualizar no Navegador
Abra o navegador e acesse:
```bash
npm install](http://localhost:3000)
```
